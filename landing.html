Summary: Latitude vs. X

  1. Home

The purpose of this project was to analyze how weather changes as you get closer to the equator. To acocomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset, we use Matplotlib to plot varios aspects of the weather vs latitude. Factors we looked at included: temperature, cloudiness, wind speed, and humudity. This site provides the source data an visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
